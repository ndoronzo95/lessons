Editor di testo

useful shortcuts (al di fuori di quelli classici di un editor di testo)

  • alt + shift + f - formatta il file
  • ctrl + p - cerca tra i file per nome
  • ctrl + d - inizia a selezionare elementi di testo uguali, uno alla volta
  • alt + ARROW_UP / ARROW_DOWN - sposta una riga di testo
  • ctrl + / - mette gli elementi selezionati in un commento e viceversa
