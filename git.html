Recap

  • git init Crea la repository git del progetto e inizia a tracciare i file
  • git add FILENAME | . Aggiunge uno o più cambiamenti alla staging area
  • git reset -- FILENAME Rimuove uno o più cambiamenti dalla staging area
  • git commit -m "Testo che descrive brevemente le mie attività" Salva i cambiamenti che erano in stagin area in un branch (timeline)
  • git status Mostra lo stato della repository: cambiamenti tracciati e in stage
  • git log Mostra lo storico del branch: tutti i commit

Recap

  • git remote add REPOSITORY_URL Collega la tua repository locale con una repository remota
  • git push Sincronizza i commit presenti in un branch della tua repository locale a quello connesso della repository remota
  • git push -u REMOTE_REPOSITORY_NAME (di solito origin) BRANCH_NAME Connette un branch in locale con un branch remoto e ne sincronizza i commit, da locale a remoto
  • git pull Sincronizza i commit presenti in un branch della tua repository remota a quello connesso della repository locale
  • git remote -v Mostra a quale repository remota è connessa la tua repository locale
  • git branch -a Mostra tutti i branch (sia locali che remoti)
  • git clone REPOSITORY_URL Scarica una repository
